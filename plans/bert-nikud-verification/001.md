# Setup and Data Verification

**Goal**: Verify the project environment is correctly set up and data can be loaded.

## Steps

1. Check Python environment and dependencies
2. Create a minimal sample dataset (10 lines)
3. Verify tokenizer works
4. Verify data loading pipeline

## Commands

```bash
# Install dependencies
uv sync

# Create sample data
cat > sample_data.txt << 'EOF'
הָאִישׁ רָצָה
הָאִישָׁה הָלְכָה
הַיְלָדִים שָׂחֲקוּ
הַמִּלָּה הִיא טוֹבָה
הַסֵּפֶר נִמְצָא
הָעִיר גְּדוֹלָה
הַשָּׁמַיִם כְּחוּלִים
הַיּוֹם יָפֶה
הַלַּיְלָה חָשֵׁךְ
הָעוֹלָם יָפֶה
EOF

# Run verification
python 001_verify_setup.py
```

## Expected Results

- Dependencies installed successfully
- Tokenizer loads correctly
- Sample data loads without errors
- NFD normalization verified
