# Model Initialization Verification

**Goal**: Verify the model architecture can be initialized and forward pass works.

## Steps

1. Initialize HebrewNikudModel
2. Verify all 5 classification heads exist
3. Run forward pass with sample data
4. Verify loss computation
5. Verify predict method

## Commands

```bash
python 002_verify_model.py
```

## Expected Results

- Model loads without errors
- All 5 classification heads present (vowel, dagesh, sin, stress, prefix)
- Forward pass returns correct output shapes
- Loss computation works
- Prediction returns valid class indices
